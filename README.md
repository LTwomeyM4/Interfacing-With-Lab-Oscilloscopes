# Interfacing-With-Lab-Oscilloscopes
Code for interfacing and getting data from Lab Oscilloscopes. 
Works with: 
1. MCP720009
2. MCP7200010
3. MCP7200011 (presumably, yet to be tested)

### To Use:
Simply Download the MP72_Def.py file and import it into any working file. Define an object of class Scope_Control and work away!
